Mise à jour du composant
========================

### Mettre les sources

### Faire l'instantané

``` bash
git add .
```
 
### Valider les changements

``` bash
git commit -m "Version 1.2.3"
```

### Créer le tag

``` bash
git tag 1.2.3 -m "Version 1.2.3"
```

### Pousser les changements sur GitHub

``` bash
git push
git push --tags
```
